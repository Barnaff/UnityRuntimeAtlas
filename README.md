# Runtime Atlas Packer

High-performance runtime texture atlas packing with dynamic updates, Burst compilation, and zero-allocation design. Perfect for dynamic sprite atlasing at runtime.

## Installation

Add the following to your Unity project's `Packages/manifest.json`:

```
"com.kobic.runtimeatlaspacker": "https://github.com/Barnaff/UnityRuntimeAtlas.git"
```

Or use the Unity Package Manager:
`Add package from Git URL...` and enter:
`https://github.com/Barnaff/UnityRuntimeAtlas.git`

## Documentation

See the [documentation](https://github.com/Barnaff/UnityRuntimeAtlas/#readme) for usage and examples.
